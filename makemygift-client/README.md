# makemygift-client

Frontend for **makemygift** — the interactive gift platform.

Project skeleton (Phase 0): Vite + React + Tailwind, booting to one styled
page. The creator studio and the experience player come in later phases.

## Run locally
```bash
npm install
cp .env.example .env   # optional
npm run dev            # http://localhost:5173
npm run build          # production build into dist/
```

## Stack
- Vite (dev server + build)
- React 19 (JavaScript)
- Tailwind CSS v4 (via @tailwindcss/vite)

## Build plan
- **Phase 0 — setup** ← you are here
- Phase 2 — experience player (plays a gift from config)
- Phase 3 — creator studio (build a gift, get a link)
- Phase 4 — media in the final reveal
